TUTORIAL HOW TO USE USGC

How To Create New Project

  1. Go To My Project Page


    2. Input Project Name

      Input Project Name to add your Project Name


    3. Input Project Name

      Input Project Description to add your Project Description Detail


    4. Press Submit Button

How To Create New Feature

  1. Go To My Project Page

      Scroll down until you see the Project List that has been created, select the Detail button to continue


    2. Select Button Add Feature

      Select the Add Feature button to go to the Add Feature page


    3. Input User Story Scenario Data

      Input Your User Story Scenario data
    • Feature Name
      Feature Name is the name of the Use Case Specification to be created
    • Action
      Action is what the User Story Scenario wants to do
    • Actor
      Actor is who the user of this User Story Scenario is
    • Features Benefits
      Feature Benefit is Final Condition in User Story Scenario


    4. Click Add User Story Scenario Condition

User Story Scenario Condition

  1. Input User Story Scenario and Condition
    • Scenario Name
      Name the scenario to find out what scenario it is
    • Given
      Defining pre-condition (actor's current location)
      example : Users are on the "Manage System Data" page
    • When
      Defining actor's action to the system
      example : Fill in the “System Name” field as the new system name
    • Then
      Defining system's responses
      example : Displays the new system name in the system data table


    2. Add Another User Story Scenario
    • The scenario's name will be the condition (ex: 'Wrong Password')
    • Condition type for alternative scenario is limited to Then.
    • Condition with other type for alternative scenario (Given, When) will be collected from normal scenario.

How to Generate

  1. Go To Page Detail Project

      Scroll down until you find a list of Features that have been created, then select the Generate action to generate the feature
